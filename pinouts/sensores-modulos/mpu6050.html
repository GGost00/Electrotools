🔍 ¿Qué es un IMU?

IMU = Inertial Measurement Unit (Unidad de Medición Inercial)

El MPU6050 combina:

  • Acelerómetro de 3 ejes: Mide aceleración en X, Y, Z (gravedad, movimiento)
  • Giroscopio de 3 ejes: Mide velocidad angular (rotación) en X, Y, Z
  • DMP (Digital Motion Processor): Procesa datos internamente para calcular orientación

Total: 6 DOF (6 Degrees of Freedom) = 6 ejes de medición

⚙️ Especificaciones Técnicas

Voltaje: 3.3V - 5V (módulo GY-521 tiene regulador LDO)
Corriente: 3.9mA en operación, 500µA en sleep mode
Protocolo: I2C (hasta 400kHz Fast Mode)
Dirección I2C: 0x68 (AD0=LOW) o 0x69 (AD0=HIGH)
Rango acelerómetro: ±2g, ±4g, ±8g, ±16g (configurable)
Rango giroscopio: ±250°/s, ±500°/s, ±1000°/s, ±2000°/s
ADC interno: 16 bits (alta precisión)
Temperatura: -40°C a +85°C
Dimensiones: 21mm x 16mm (módulo GY-521)

🔌 Dirección I2C (Pin AD0)

El MPU6050 puede tener 2 direcciones I2C diferentes:

  • AD0 conectado a GND (o flotante): Dirección = 0x68
  • AD0 conectado a VCC: Dirección = 0x69

Esto permite usar 2 MPU6050 en el mismo bus I2C.

¿Cómo verificar? Usa el I2C Scanner de Arduino para detectar la dirección activa.

⚠️ Resistencias Pull-Up I2C

El protocolo I2C requiere resistencias pull-up de 4.7kΩ en SDA y SCL.

¿Debo agregarlas?

  • ✅ Módulo GY-521: Ya incluye pull-ups en la PCB (no necesitas agregar)
  • ⚠️ Chip MPU6050 solo: Debes agregar 2 resistencias de 4.7kΩ (SDA→VCC, SCL→VCC)
  • ✅ Arduino Uno: También tiene pull-ups internas débiles (puede funcionar sin externas)

Problema: Si tienes múltiples dispositivos I2C, demasiadas pull-ups pueden causar problemas (usar 2.2kΩ en lugar de 4.7kΩ).

💻 Código Arduino Básico

#include <Wire.h>
#include <MPU6050.h>

MPU6050 mpu;

void setup() {
  Serial.begin(9600);
  Wire.begin();
  
  Serial.println("Inicializando MPU6050...");
  mpu.initialize();
  
  if (mpu.testConnection()) {
    Serial.println("MPU6050 conectado!");
  } else {
    Serial.println("Error de conexión");
  }
}

void loop() {
  int16_t ax, ay, az;
  int16_t gx, gy, gz;
  
  mpu.getMotion6(&ax, &ay, &az, &gx, &gy, &gz);
  
  Serial.print("Accel: ");
  Serial.print(ax); Serial.print(" ");
  Serial.print(ay); Serial.print(" ");
  Serial.print(az); Serial.print(" | Gyro: ");
  Serial.print(gx); Serial.print(" ");
  Serial.print(gy); Serial.print(" ");
  Serial.println(gz);
  
  delay(100);
}
