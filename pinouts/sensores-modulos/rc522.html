🔍 ¿Qué es RFID?

RFID = Radio-Frequency Identification (Identificación por Radiofrecuencia)

Componentes:

  • Lector RC522: Genera campo electromagnético de 13.56MHz
  • Tarjeta/Tag RFID: Contiene chip y antena (pasivo, sin batería)
  • Funcionamiento: Campo del lector alimenta el chip → Chip envía UID → Lector captura datos

UID: Unique Identifier (código único de cada tarjeta, típicamente 4 bytes = 8 dígitos hexadecimales)

⚙️ Especificaciones Técnicas

Voltaje: 3.3V DC (⚠️ NO tolerante a 5V)
Corriente: 13-26mA en operación, 10-13mA en standby
Protocolo: SPI (hasta 10Mbps)
Frecuencia: 13.56MHz (HF - High Frequency)
Distancia de lectura: 0 - 6cm (depende de antena)
Tarjetas compatibles: Mifare Classic 1K, Mifare Classic 4K, Mifare Ultralight, NTAG213/215/216
Velocidad de lectura: <100ms
Temperatura operativa: -20°C a +80°C
Dimensiones: 40mm x 60mm (módulo típico)

⚠️ ADVERTENCIA CRÍTICA: 3.3V ÚNICAMENTE

🔴 NUNCA conectes el RC522 a 5V - El chip MFRC522 funciona a 3.3V y NO es tolerante a 5V.

¿Qué pasa si conecto a 5V? El módulo se dañará permanentemente (el regulador interno no soporta 5V en la entrada).

Solución Arduino Uno: Usa el pin 3.3V del Arduino (no 5V). El pin 3.3V puede entregar hasta 50mA, suficiente para el RC522.

Problema de los pines SPI: Los pines MOSI, SCK del Arduino Uno son 5V, pero el RC522 tiene resistencias pull-up internas que protegen las entradas (funciona sin problemas).

📋 Tipos de Tarjetas Mifare

Mifare Classic 1K: 1024 bytes de memoria (16 sectores × 4 bloques × 16 bytes). Más común y económica. UID de 4 bytes.

Mifare Classic 4K: 4096 bytes de memoria. Mayor capacidad para aplicaciones complejas. UID de 4 bytes.

Mifare Ultralight: 64 bytes. Sin autenticación (menos segura). Ideal para tickets desechables.

NTAG213/215/216: Compatible NFC (Near Field Communication). Funciona con smartphones Android. Capacidad: 144/504/888 bytes.

💻 Código Arduino Básico

#include <SPI.h>
#include <MFRC522.h>

#define SS_PIN 10
#define RST_PIN 9

MFRC522 rfid(SS_PIN, RST_PIN);

void setup() {
  Serial.begin(9600);
  SPI.begin();
  rfid.PCD_Init();
  
  Serial.println("Acerca tarjeta RFID...");
}

void loop() {
  if (!rfid.PICC_IsNewCardPresent())
    return;
    
  if (!rfid.PICC_ReadCardSerial())
    return;
  
  Serial.print("UID: ");
  for (byte i = 0; i < rfid.uid.size; i++) {
    Serial.print(rfid.uid.uidByte[i] < 0x10 ? " 0" : " ");
    Serial.print(rfid.uid.uidByte[i], HEX);
  }
  Serial.println();
  
  rfid.PICC_HaltA();
  rfid.PCD_StopCrypto1();
}

📚 Librería MFRC522

Instalación: Arduino IDE → Sketch → Include Library → Manage Libraries → Buscar "MFRC522"

Autor: Miguel Balboa (GithubCommunity)

Funciones principales:

  • PCD_Init() - Inicializar lector
  • PICC_IsNewCardPresent() - Detectar tarjeta
  • PICC_ReadCardSerial() - Leer UID
  • MIFARE_Read(block, buffer, size) - Leer bloque de memoria
  • MIFARE_Write(block, buffer, size) - Escribir bloque

Ejemplos: File → Examples → MFRC522 → DumpInfo (ver contenido completo de tarjeta)

🎯 Aplicaciones Comunes

  • Control de acceso: Cerraduras electrónicas, puertas automáticas
  • Sistemas de asistencia: Registro de entrada/salida de empleados
  • Pagos sin contacto: Tarjetas de transporte público, comedores
  • Inventario: Seguimiento de productos con etiquetas RFID
  • Domótica: Activar luces, música, sistemas al detectar usuario
  • Clonación de tarjetas: Copiar tarjetas duplicables (ética cuestionable)
  • NFC tags: Programar acciones en smartphones Android (NTAG)
