🌡️ DHT11 / DHT22

Sensor Digital de Temperatura y Humedad

🌡️

[ESPACIO PARA IMAGEN: dht11.png]

Diagrama de pinout DHT11/DHT22

📌 Pinout DHT11 (4 pines)

Pin Nombre Función Notas
1 VCC Alimentación 3.3V - 5V DC
2 DATA Datos digitales Protocolo propietario 1-wire
3 NC No conectado Sin uso (solo DHT11 de 4 pines)
4 GND Tierra Ground común con microcontrolador

📌 Pinout DHT11 en Módulo (3 pines)

Pin Etiqueta Función Conexión Arduino
1 VCC / + Alimentación 3.3V - 5V 5V (Arduino Uno) o 3.3V (ESP32)
2 DATA / OUT / S Señal de datos digital Cualquier pin digital (ej: D2)
3 GND / - Tierra común GND Arduino

🔍 DHT11 vs DHT22

DHT11: Rango de temperatura 0-50°C (±2°C), humedad 20-80% (±5%). Económico y de baja precisión.

DHT22 (AM2302): Rango -40 a 80°C (±0.5°C), humedad 0-100% (±2%). Mayor precisión y rango, más costoso.

Ambos usan el mismo protocolo y librería (DHT.h en Arduino).

⚙️ Especificaciones Técnicas

Voltaje: 3.3V - 5.5V DC
Corriente: 0.5 - 2.5mA medición, 100-150µA en espera
Protocolo: 1-Wire propietario (no compatible con Dallas 1-Wire)
Tiempo de muestreo: 1 lectura cada 2 segundos (DHT11) o 1 lectura cada 1 segundo (DHT22)
Dimensiones: 15.5mm x 12mm x 5.5mm (DHT11)

🔌 Resistencia Pull-Up

El pin DATA requiere una resistencia pull-up de 4.7kΩ - 10kΩ conectada entre DATA y VCC. Muchos módulos DHT11 ya incluyen esta resistencia integrada en la PCB.

Verificación: Si el módulo tiene 3 pines, probablemente ya incluye pull-up. Si usa el sensor de 4 pines directo, debes agregar la resistencia externa.

⚠️ Advertencias Importantes

Tiempo de estabilización: Espera 1-2 segundos después de encender antes de leer datos.

Intervalo mínimo: No leas el sensor más rápido que 1 vez por segundo (DHT22) o 2 segundos (DHT11). Lecturas rápidas causan errores.

Longitud de cable: Máximo 20-30 metros entre sensor y microcontrolador. Cables largos requieren resistencia pull-up menor (3.3kΩ).

💻 Código Arduino Básico

#include <DHT.h>

#define DHTPIN 2        // Pin DATA del DHT11
#define DHTTYPE DHT11   // o DHT22

DHT dht(DHTPIN, DHTTYPE);

void setup() {
  Serial.begin(9600);
  dht.begin();
}

void loop() {
  float h = dht.readHumidity();
  float t = dht.readTemperature();
  
  if (isnan(h) || isnan(t)) {
    Serial.println("Error leyendo DHT!");
    return;
  }
  
  Serial.print("Humedad: ");
  Serial.print(h);
  Serial.print(" %  Temp: ");
  Serial.print(t);
  Serial.println(" °C");
  
  delay(2000); // Esperar 2 seg
}

📚 Librerías Necesarias

Arduino: DHT sensor library by Adafruit (incluye dependencia Adafruit Unified Sensor)

Instalación: Arduino IDE → Sketch → Include Library → Manage Libraries → Buscar "DHT sensor library"

ESP32/ESP8266: Compatible con la misma librería DHT.h de Adafruit

Raspberry Pi: Librería Adafruit_CircuitPython_DHT (Python)

🎯 Aplicaciones Comunes

  • Estaciones meteorológicas: Monitoreo de temperatura y humedad ambiente
  • Domótica: Control de humidificadores, deshumidificadores, sistemas HVAC
  • Invernaderos: Automatización de riego según humedad ambiental
  • Incubadoras: Control de temperatura para huevos o cultivos
  • Dataloggers: Registro de condiciones ambientales en tiempo real
  • Almacenes: Monitoreo de condiciones de almacenamiento de productos sensibles
