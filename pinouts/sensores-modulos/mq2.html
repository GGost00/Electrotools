💨 MQ-2 Sensor de Gas

Detector Analógico de Gases Inflamables y Humo

💨

[ESPACIO PARA IMAGEN: mq2.png]

Diagrama de pinout MQ-2

📌 Pinout MQ-2 Módulo Digital (4 pines)

Pin Etiqueta Función Conexión Arduino
1 VCC Alimentación 5V (consume ~150mA en calentamiento)
2 GND Tierra GND Arduino
3 DOUT (DO) Salida digital Cualquier pin digital (ej: D8) - HIGH cuando supera umbral
4 AOUT (AO) Salida analógica Cualquier pin analógico (ej: A0) - Lectura 0-1023

🔧 Controles del Módulo

Control Función Efecto
Potenciómetro Ajuste de sensibilidad Controla el umbral para activar salida digital DOUT. Girar horario = menos sensible (requiere más gas)
LED de Power Indicador de alimentación Enciende cuando el módulo recibe 5V
LED de Detección Indicador de gas detectado Enciende cuando DOUT está en HIGH (gas supera umbral)

🔍 Gases Detectables por MQ-2

El MQ-2 puede detectar múltiples gases mediante cambios en resistencia de su elemento sensor (óxido de estaño SnO2):

  • GLP (Gas Licuado de Petróleo): 200-5000 ppm
  • Propano: 200-5000 ppm
  • Butano: 300-5000 ppm
  • Metano (gas natural): 5000-20000 ppm
  • Hidrógeno: 300-5000 ppm
  • Humo: Detección cualitativa (no cuantitativa)
  • Alcohol: 100-2000 ppm

⚠️ No es específico: Reacciona a varios gases simultáneamente. Para detección específica usa MQ-3 (alcohol), MQ-4 (metano), MQ-7 (CO), etc.

⚙️ Especificaciones Técnicas

Voltaje: 5V DC (±0.1V)
Corriente: 150mA en precalentamiento, 60-150mA en operación
Voltaje de calentamiento: 5.0V ± 0.1V (resistencia de calentamiento interna)
Resistencia de carga: 20kΩ (ajustable)
Rango de detección: 200-10000 ppm (partes por millón)
Tiempo de respuesta: <10s
Tiempo de precalentamiento: 24-48 horas ideal, mínimo 20 segundos
Temperatura operativa: -10°C a +50°C
Humedad: <95% RH (Relative Humidity)
Vida útil: >5 años

🔥 Tiempo de Precalentamiento (Preheat)

⚠️ CRÍTICO: El sensor MQ-2 requiere calentamiento del elemento sensor (resistencia de SnO2) para funcionar correctamente.

Precalentamiento mínimo: 20-30 segundos (lecturas inestables)

Precalentamiento recomendado: 24-48 horas continuas para estabilización completa y calibración en aire limpio

Durante el precalentamiento:

  • El sensor se calienta a ~200-300°C
  • Lecturas analógicas varían aleatoriamente
  • El módulo puede sentirse tibio al tacto (normal)

✅ Solución: Ignora las primeras lecturas durante 60 segundos en tu código después del setup().

⚠️ Limitaciones y Advertencias

No es un medidor calibrado: Valores analógicos son relativos, no absolutos. No confíes en ppm exactos sin calibración profesional.

Humedad afecta lecturas: Vapor de agua puede activar falsas alarmas (baño después de ducha).

Deriva térmica: Cambios de temperatura ambiente alteran las lecturas (requiere recalibración).

No apto para espacios confinados: No es sensor certificado para uso industrial en áreas peligrosas (ATEX/IECEx).

Consumo alto: 150mA puede agotar baterías rápidamente (no ideal para proyectos a batería).

Ventilación necesaria: Coloca el sensor en área ventilada, no en caja cerrada.

💻 Código Arduino Básico

#define MQ2_AOUT A0
#define MQ2_DOUT 8
#define BUZZER 9

void setup() {
  Serial.begin(9600);
  pinMode(MQ2_DOUT, INPUT);
  pinMode(BUZZER, OUTPUT);
  
  Serial.println("Calentando sensor MQ-2...");
  delay(20000); // Esperar 20s mínimo
  Serial.println("Sensor listo!");
}

void loop() {
  int analogValue = analogRead(MQ2_AOUT);
  int digitalValue = digitalRead(MQ2_DOUT);
  
  Serial.print("Gas analógico: ");
  Serial.print(analogValue);
  Serial.print(" | Digital: ");
  Serial.println(digitalValue);
  
  if (digitalValue == HIGH) {
    Serial.println("¡ALERTA GAS DETECTADO!");
    digitalWrite(BUZZER, HIGH);
  } else {
    digitalWrite(BUZZER, LOW);
  }
  
  delay(1000);
}

📊 Interpretación de Valores Analógicos

En aire limpio: Lectura típica ~100-200 (valor bajo)

Presencia de gas ligera: 300-500 (gas detectado pero seguro)

Concentración media: 500-700 (activar ventilación)

Concentración peligrosa: >700 (activar alarma, evacuar)

⚠️ Nota: Valores varían según calibración del potenciómetro, temperatura ambiente, humedad. Estos son aproximados.

Calibración: En aire limpio, ajusta potenciómetro hasta que LED de detección esté apagado. Luego, acerca gas ligero (encendedor sin encender) para probar umbral.

🎯 Aplicaciones Comunes

  • Alarma de fugas de gas: Detectar GLP en cocinas, calefones, calderas
  • Detector de humo casero: Alarma temprana de incendios (complemento a detector de humo certificado)
  • Monitoreo de garajes: Detectar monóxido de carbono (CO) de vehículos (mejor usar MQ-7 específico CO)
  • Sistemas de ventilación automática: Activar extractor cuando detecta gas
  • Proyectos educativos: Aprender sobre sensores químicos y ADC
  • Alcoholímetro casero: Detección de alcohol en aliento (usar MQ-3 es más preciso)

⚠️ ADVERTENCIA: Este sensor NO sustituye detectores de gas certificados para uso residencial/industrial. Úsalo solo con fines educativos o como complemento.
