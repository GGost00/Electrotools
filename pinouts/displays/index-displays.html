LCD 16x2 Paralelo

Display de caracteres alfanuméricos con interfaz paralela de 4 u 8 bits

Tipo: Caracteres alfanuméricos

Tamaño: 16 columnas × 2 filas

Controlador: HD44780 o compatible

Interfaz: Paralelo 4/8 bits

Voltaje: 5V

Pines: 16 (14 + backlight)

LCD con módulo I2C

Interfaz: I2C (2 cables + alimentación)

📺 Tipos de Displays

Los displays se clasifican por tecnología (LCD, OLED, TFT) y tipo de interfaz (paralelo, I2C, SPI, UART). La elección depende de requisitos de resolución, consumo, número de pines disponibles y velocidad de actualización.

🔌 Interfaces Comunes

I2C: Solo 2 pines (SDA/SCL), velocidad moderada. SPI: Mayor velocidad, requiere más pines. Paralelo: Muy rápido pero consume muchos pines. UART: Simple para displays inteligentes.

⚡ Alimentación

La mayoría de displays LCD requieren 5V, mientras que OLEDs pequeños funcionan con 3.3V o 5V. Verifica siempre el voltaje antes de conectar. Algunos displays incluyen reguladores integrados.
