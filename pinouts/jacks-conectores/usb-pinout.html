📋 Tipos de Conectores USB

USB Type-A

Uso: Host (computadoras, cargadores)

Pines: 4 (USB 2.0), 9 (USB 3.0)

Forma: Rectangular, conexión unidireccional

USB Type-B / Mini / Micro

Type-B: Impresoras, scanners (cuadrado)

Mini-B: Cámaras antiguas (obsoleto)

Micro-B: Smartphones Android (pre-2017)

USB Type-C

Reversible: No tiene orientación

Pines: 24 contactos

Velocidad: Hasta 40 Gbps (USB4)

Power Delivery: Hasta 240W

Versiones USB

USB 1.1: 12 Mbps (1998)

USB 2.0: 480 Mbps (2000)

USB 3.0/3.1: 5/10 Gbps (azul)

USB 4.0: 40 Gbps (2019)

⚡ USB Power Delivery (PD)

USB-C con PD permite negociar voltaje (5V, 9V, 12V, 15V, 20V, 28V, 36V, 48V) y corriente (hasta 5A). Esto permite cargar laptops y dispositivos de alta potencia. Requiere chip controlador (FUSB302, STUSB4500).

🔄 USB OTG (On-The-Go)

Micro-USB con pin ID (pin 5) permite que un dispositivo actúe como host o device. Si ID está a GND, actúa como host (puede conectar pendrives, teclados). Si ID está flotante, actúa como device.

🎨 Identificación Visual

USB 2.0: Plástico blanco/negro. USB 3.0: Plástico azul (Type-A). USB 3.1: Plástico teal/turquesa. USB-C: Reversible, pequeño, ovalado.

📊 Cables USB

No todos los cables USB son iguales. Cables solo carga no tienen pines D+/D- conectados. Cables USB 3.0 necesitan más hilos (9 vs 4). USB-C full-featured tiene 24 hilos (algunos cables baratos solo tienen 4).
